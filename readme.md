# Apply custom theme

1. ZSH Custom theme is saved as `'oh my zsh anonymous.zsh-theme'`.
2. Rename the file as `anonymous.zsh-theme` and add it to `~/.oh-my-zsh/custom/themes/`.
3. Open `~/.zshrc` and change the `ZSH-THEME=[THEME_NAME]` to `ZSH-THEME="anonymous"` and save the file.
> IF YOU WANT YOU CAN USE MY ZSHRC FILE WHICH CAN BE FOUND IN THIS REPO
4. Run this command to enable the theme `clear; source ~/.zshrc`.
